# 📡 Trend Pulse AI — Intelligent Google Trends Dashboard

Trend Pulse AI is a powerful **Streamlit-based analytics dashboard** that allows users to explore, compare, and analyze **Google Trends data in real-time**.

It goes beyond basic visualization by providing **AI-driven insights, trend analysis, and predictive intelligence**, making it a smart tool for researchers, marketers, and data enthusiasts.

---

## 🚀 Features

### 🔍 Core Features

* Compare up to **5 keywords simultaneously**
* Real-time data using **Google Trends (pytrends)**
* Interactive **Plotly visualizations**
* Multiple time ranges and region selection
* CSV data export support

---

### 📊 Advanced Analytics

* 📈 Interest over time (Time Series)
* 🌍 Geographic distribution (Heatmaps)
* 🔗 Correlation analysis between keywords
* 📉 Rolling averages and smoothing
* 📦 Distribution analysis (Box plots)

---

### 🤖 AI-Powered Features

* 🔮 **Trend Prediction** (forecast future interest)
* 🧠 **AI Insight Generator** (automatic trend explanations)
* ⚠️ **Trend Alert System** (detect sudden spikes/drops)
* 🎯 **Keyword Battle Mode** (compare performance metrics)

---

### 🔥 Trend Exploration

* Top trending searches
* Rising queries
* Related keyword discovery

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Data Source:** Google Trends via pytrends
* **Visualization:** Plotly
* **Backend:** Python
* **Data Processing:** Pandas

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/trend-pulse-ai.git
cd trend-pulse-ai
```

---

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```
streamlit run trends_app.py
```

Then open your browser at:

```
http://localhost:8501
```

---

## 🧠 How It Works

1. Enter up to **5 keywords**
2. Select:

   * Time range
   * Region
   * Category
3. Click **Analyze Trends**
4. Explore multiple sections:

   * 📈 Time Series Analysis
   * 🌍 Geographic Insights
   * 🔍 Related Queries
   * 📊 Advanced Analytics
   * 🔥 Trending Searches

---

## 📊 Data Insights

* Data is **normalized (0–100)**
* 100 represents **peak popularity**
* Data is cached for **faster performance**
* Trends reflect **relative search interest**
